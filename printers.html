Windows Setup

Download Windows Driver
  1. Run the installer and accept the terms.
  2. Select Wireless or Ethernet.
  3. Wait for timeout → click OK when "printer can’t be found".
  4. Select Add printer manually.
  5. Enter the printer IP from the list below.
  6. Type the model (e.g. 3070) in the Filter box and select the model.
  7. Click Next twice.
  8. Use a simple friendly name.

macOS Setup

Download macOS Driver
  • Choose the LPD protocol.
  • Type ff in the driver field and select FujiFilm driver.
  • Do not choose model-specific drivers.
  • Use a simple friendly name.
