Windows Setup

  1. Run the installer and accept the terms.
  2. Select Wireless or Ethernet.
  3. Wait for timeout → click OK.
  4. Select Add printer manually.
  5. Enter the printer IP.
  6. Search model in the Filter.
  7. Click Next twice.
  8. Name it something simple.

macOS Setup

  • Use the LPD protocol.
  • Type ff in the driver field.
  • Select FujiFilm driver.
  • Name the printer something simple.
