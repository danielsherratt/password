Windows Setup

  1. Run installer and accept terms.
  2. Select Wireless/Ethernet.
  3. Let it time out → press OK.
  4. Choose Add printer manually.
  5. Enter IP address.
  6. Type model into Filter.
  7. Click Next twice.
  8. Give a simple name.

macOS Setup

  • Use LPD protocol.
  • Type "ff" in driver box.
  • Select FujiFilm driver.
  • Name printer simply.
