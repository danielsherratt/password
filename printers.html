Printer Setup Help

Windows Printer Setup

First, check if you are using Papercut Hive:

  • If Papercut Hive is already set up and this is a new or re-imaged laptop, search for the Papercut invite email.
  • If the “Get started” link says it is expired, click New Invite and wait up to 10 minutes.
  • If you are not using Papercut Hive, choose your school or site from the list below.

Install FujiFilm Driver (Windows)

Download FujiFilm Windows Driver

Installation Steps:

  1. Run the installer and accept the terms.
  2. Select Wireless or Ethernet connection.
  3. Wait for timeout → click OK when "printer can’t be found".
  4. Choose Add printer manually.
  5. Enter the printer IP (from the list below).
  6. In the Filter box, type the model (e.g. 3070) and select the correct FujiFilm model.
  7. Click Next twice.
  8. Give the printer a simple name (avoid model codes).

macOS Printer Setup

Install FujiFilm Driver (macOS)

Download FujiFilm macOS Driver

When adding the printer in System Settings:

  • Choose the LPD protocol.
  • For Driver, type “ff” and select the FujiFilm driver.
  • Do not select a model-specific driver.
  • Use a simple friendly printer name.
