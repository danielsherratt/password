Printer Setup Help

Windows Printer Setup

Install FujiFilm Driver (Windows)

Download FujiFilm Windows Driver
  1. Run the installer and accept the terms.
  2. Select Wireless or Ethernet connection.
  3. Wait for timeout → click OK when "printer can’t be found".
  4. Select Add printer manually.
  5. Enter the printer IP from the list below.
  6. Type the model (e.g. 3070) into the Filter box and select your model.
  7. Click Next twice.
  8. Name the printer something simple (avoid model codes).

macOS Printer Setup

Install FujiFilm Driver (macOS)

Download FujiFilm macOS Driver
  • When adding the printer, select the LPD protocol.
  • In the driver field, type ff and choose the FujiFilm driver.
  • Do not select a model-specific driver.
  • Use a simple friendly printer name.
