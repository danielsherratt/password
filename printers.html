Written Instructions

Windows Setup

  1. Run the installer and accept the terms.
  2. Select Wireless or Ethernet.
  3. Wait for timeout → click OK when "printer can’t be found".
  4. Select Add printer manually.
  5. Enter the printer IP from the list below.
  6. Use the model in the Filter box.
  7. Click Next twice.
  8. Name it something simple.

macOS Setup

  • Select the LPD protocol.
  • Type ff in the driver field.
  • Select FujiFilm driver.
  • Name the printer something simple.
